Search posts/pages/post_types for Slug in Admin panel only.

== Description ==
This plugin allows you to search post/pages/post_types by slug inside /wp-admin area.
Examples:
When you search for “slug:about”  it will return results with all posts which have “about” in their slug.

== Installation ==
Copy the "WP_Search_Slug Plugin" folder to your WordPress Plugin folder (wp-content/plugins) and activate it.
